## Overriding

You can apply the desired properties and the rest will be combined

```javascript
import { useAuth, useLayout, useOutsideClick, useDisableScroll, useInterval, useScroll } from 'quantic-ui/hooks'

// get user informations
const { email, isAuthenticated, signIn, signOut } = useAuth()

// get app informations
const { navbarIsFixed, theme, language, setTheme, setLanguage, onChangeTheme, isMobile, appName } = useLayout()

// based on ref, execute callback action
useOutsideClick(ref, callback)

// disabled scroll based on parametter
useDisableScroll(true)

// execute callback after delay
useInterval(callback, delay)

const { scrollY, scrollX, scrollDirection } = useScroll()
```
